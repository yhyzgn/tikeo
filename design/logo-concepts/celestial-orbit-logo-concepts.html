Tikeo celestial logo studies

天体物理运动 + 呼吸律动 + 主题联动

这只是概念预览，不会替换站点现有 Logo。三个方案都用开普勒椭圆轨道求解位置：公转角速度不匀速，近拱点更快；天体自转独立匀角速度。呼吸只作用于光晕与能量环，不篡改轨道。

方案 A · 双星调度核

稳定、公转、自转、近拱点加速；最接近当前“呼吸核心”的精神延展。
适合：文档站 / 产品主 Logo。中心代表调度核，双天体代表 Server 与 Worker 的稳定协同。
色调：暗色用冰蓝 + 星紫；亮色自动转为清晰蓝 + 紫。保留科技感但不廉价霓虹。

方案 B · 共振队列环

2:3:5 轨道周期，表达任务队列、重试与实例状态的节律。
适合：更强调任务编排、状态流转和通知触达的品牌方向。
动态：三颗行星严格沿各自可见椭圆轨道运行；每颗行星带一个本地卫星，中心恒星做克制呼吸。

方案 C · 引力隧道门

倾斜轨道 + 透视门，偏企业级、基础设施、Worker Tunnel。
适合：如果后续更想把 Tikeo 定位为“企业内网 Worker 出站控制平面”。
识别：中心门洞有基础设施感；天体围绕隧道口运动，呼吸环像连接心跳。
